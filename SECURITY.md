## Reporting a Vulnerability

We take the security of CtrlHelp seriously. If you discover a security vulnerability, please follow these steps to report it:

1.  **Do not create a public issue.** Security vulnerabilities should be handled discreetly to protect users.
2.  Send an email to [security@example.com] (replace with actual contact or generic instruction) with a description of the vulnerability.
3.  Include steps to reproduce the issue.
4.  We will acknowledge your report within 48 hours and provide patches or workarounds as soon as possible.
